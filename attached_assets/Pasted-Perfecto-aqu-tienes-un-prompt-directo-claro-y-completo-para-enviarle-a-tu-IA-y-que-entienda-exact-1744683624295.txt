Prompt — Eliminar gráfico actual y rehacerlo correctamente

Mia, necesito que elimines completamente el gráfico tipo donut que aparece en la sección “Gastos por Categoría” y lo vuelvas a hacer desde cero, bien alineado, con un diseño limpio y equilibrado.

¿Qué está mal?

Ahora mismo el gráfico:
	•	Está desalineado, demasiado pegado al margen izquierdo.
	•	No guarda proporción ni equilibrio con el resto del contenido.
	•	Rompe el diseño visual tipo Apple que buscamos.

¿Qué quiero exactamente?
	1.	Elimina por completo el gráfico actual.
	2.	Reemplázalo por uno centrado horizontal y verticalmente dentro de su contenedor.
	3.	Debe quedar bien balanceado con la lista de categorías a la derecha del gráfico.
	4.	El diseño tiene que ser limpio, ordenado, sin márgenes rotos ni contenido descompensado.
	5.	Que se mantenga la estética Apple: amplitud, simplicidad, buena jerarquía visual.

Estructura visual que necesito:

-----------------------------------------
|            Gastos por Categoría       |
|                                       |
|   [       DONUT CHART       ]   [Lista de categorías con iconos, importes y porcentajes]  |
|                                       |
-----------------------------------------

Ejemplo de código con Tailwind CSS (puedes adaptarlo):

<div class="flex items-center justify-between p-6">
  <!-- Donut chart -->
  <div class="w-1/2 flex justify-center">
    <canvas id="gastosDonutChart" class="max-w-[200px]"></canvas>
  </div>

  <!-- Lista de categorías -->
  <div class="w-1/2 pl-6 space-y-4">
    <div class="flex items-center gap-2">
      <div class="w-4 h-4 rounded-full bg-[#E74C3C]"></div>
      <span class="text-sm font-medium">Oficina - 51.730 € (99.55%)</span>
    </div>
    <div class="flex items-center gap-2">
      <div class="w-4 h-4 rounded-full bg-[#F4B400]"></div>
      <span class="text-sm font-medium">Suministros - 235 € (0.45%)</span>
    </div>
  </div>
</div>

Consideraciones adicionales:
	•	Asegúrate de que el contenedor no tenga márgenes rotos ni padding mal distribuido.
	•	Usa Chart.js o lo que estés usando, pero asegurándote que el gráfico se renderice en el centro de su contenedor, no alineado al borde.
	•	El resultado tiene que parecer una app premium, como si fuera diseñada por Apple.

Hazlo desde cero y asegúrate de que no se repita el error de desalineación.
